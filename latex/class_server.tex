\doxysubsection{Detailed Description}
Класс \doxylink{class_server}{Server} представляет собой логику сервера. 

Этот класс обеспечивает авторизацию, регистрацию, отправку заданий, проверку заданий, управление подключениями и другие функции, связанные с обработкой запросов от клиентов.
